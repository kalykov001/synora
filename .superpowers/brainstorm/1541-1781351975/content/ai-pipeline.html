AI Pipeline — Supabase Edge Functions

Раздел 3: Как работает обработка PDF и AI-функции

Шаг 1 — Загрузка PDF
📱 Пользователь выбирает PDF
→
Supabase Storage
pdfs/{user_id}/{file}
→
documents table
status: 'processing'
Шаг 2 — Извлечение текста (Edge Function: process-pdf)
1. Скачивает PDF из Storage
2. Извлекает текст (pdf-parse / pdfjs-dist)
3. Определяет язык документа (langdetect)
4. Сохраняет текст + язык в documents
5. Обновляет status → 'ready'
Шаг 3 — Генерация контента (4 Edge Functions)
generate-summary
PDF text → Gemini Flash → конспект на нужном языке
generate-flashcards
PDF text → Gemini Flash → N пар вопрос/ответ
generate-exam
PDF text → Gemini Flash → MCQ вопросы с ответами
tutor-chat
История + PDF контекст → Gemini Pro → ответ
Модели Gemini
gemini-1.5-flash
Конспекты, flashcards, экзамены
Дёшево, быстро
gemini-1.5-pro
AI Tutor чат
Умнее, чуть дороже
Многоязычность
Каждый Edge Function принимает параметр output_language.
Промпт содержит инструкцию отвечать на нужном языке.
Поддержка: EN · RU · KY · KZ · TR · AR · DE · FR · ES
