Что строим прямо сейчас?

A

Только мобильное приложение

React Native + Expo + TypeScript. AI и backend — заглушки/моки. Полный UI, навигация, все экраны. Запускается сразу на устройстве.

+

  • Быстро
  • Можно показать инвесторам/пользователям
  • Легко итерировать дизайн

−

  • AI не работает реально
  • Backend — отдельная задача позже

Приложение + Supabase + AI API

React Native + Expo + Supabase (auth, БД, storage) + Claude/OpenAI API для AI-функций. Реально работающий продукт.

+

  • Работающий MVP
  • Supabase — быстрый бэкенд без сервера
  • Реальный AI

−

  • Нужны API-ключи
  • Больше настройки
C

Полный стек с кастомным бэкендом

React Native + Node.js/Express backend + PostgreSQL + AI. Максимальный контроль, но значительно больше работы.

+

  • Полный контроль
  • Масштабируемость

−

  • Долго
